前言：复习并总结一下java泛型

概念

泛型是JavaSE1.5的新特性，泛型的本质是参数化类型，也就是说所操作的数据类型被指定为一个参数。这种参数类型可以用在类、接口和方法的创建中，分别称为泛型类、泛型接口、泛型方法。Java语言引入泛型的好处是安全简单。

一般泛型命名

  • E - Element (在集合中使用，因为集合中存放的是元素)
  • T - Type（Java 类）
  • K - Key（键）
  • V - Value（值）
  • N - Number（数值类型）

泛型标识

使用泛型必须要声明泛型标识

Java

类名后声明的泛型，所有非静态方法都能使用声明的泛型标识
